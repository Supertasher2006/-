# Публичный запуск (Render)

## 1) Подготовка

1. Создай репозиторий на GitHub.
2. Залей в него содержимое папки `сайт_журнала`.

## 2) Создание сервиса

1. Зайди в [Render Dashboard](https://dashboard.render.com/).
2. Нажми **New +** -> **Blueprint**.
3. Выбери свой GitHub-репозиторий.
4. Render подхватит `render.yaml` автоматически.
5. Подтверди создание сервиса.

## 3) Что настраивается автоматически

- веб-сервис Python;
- запуск через `gunicorn`;
- секретный ключ сессий;
- постоянный диск `/var/data` для:
  - БД SQLite (`/var/data/tramplin.db`);
  - загруженных медиа (`/var/data/uploads`).

## 4) Результат

После деплоя Render выдаст публичный URL вида:

- `https://trampolin-journal.onrender.com`

По этому адресу сайт будет доступен с любых устройств.

## 5) Важно

- Первый запрос на бесплатном тарифе может открываться дольше (cold start).
- При изменениях в GitHub Render будет пересобирать сайт автоматически.
